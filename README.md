# The dimensionality of plant–plant competition

*Daniel B. Stouffer, Oscar Godoy, Guilio V. Dalla Riva, Margaret M. Mayfield*

## Data

The `data` directory contains the data sets analyzed in the manuscript.

## Code

The `code` directory includes all code needed to replicate the results in the manuscript and generate all figures. The `README.md` file therein explains further details about the different subdirectories.

## Fits

The `fits` directory contains the output of fits of the model to _Raw_ empirical data and to synthetic data as a result of the model-fitting code.

## Results

The `results` directory contains the output of post-processing of model fits to _Raw_ empirical data or direct outputs from the analysis of _Matrix_ empirical data.

## Figures

The `figures` directory and its subdirectories are placeholders for where all figures will be stored once created using the figure generation code.
